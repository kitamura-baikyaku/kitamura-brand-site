# 北村　充ブランドサイト Ver.6

大阪市の不動産売却相談に特化したブランドサイトです。
Next.js + Netlifyで公開する前提の構成です。

## 反映方法
1. ZIPを解凍
2. 中身を `kitamura-brand-site` フォルダへ上書きコピー
3. GitHub Desktopで `Ver6 制作会社品質へ再設計` と入力
4. Commit to main
5. Push origin
