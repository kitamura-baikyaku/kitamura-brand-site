# 反映手順

## 一括反映
1. `kitamura_brand_v7_release.zip` をダウンロード
2. 右クリック → すべて展開
3. 展開したフォルダの中身をすべてコピー
4. `ドキュメント > GitHub > kitamura-brand-site` を開く
5. 既存ファイルに上書き貼り付け
6. GitHub Desktopを開く
7. 左側に変更ファイルが表示される
8. Summaryに `Ver7.0 Release` と入力
9. `Commit to main`
10. `Push origin`
11. Netlifyでデプロイ完了を待つ

## 7段階で反映する場合
`phases` フォルダ内の `01_top` から順番にREADMEを確認してください。
ただし、通常は一括反映で問題ありません。
