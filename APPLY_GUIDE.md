# Ver.8.02 反映手順

1. ZIPを解凍します。
2. 解凍した中身を、GitHub Desktopで管理している `kitamura-brand-site` フォルダへ上書きコピーします。
3. GitHub Desktopを開きます。
4. Summaryに以下を入力します。

```text
Ver8.02 レイアウト崩れ再修正
```

5. `Commit to main` を押します。
6. `Push origin` を押します。
7. Netlifyのデプロイ完了後、以下を確認します。

- トップページ：`https://boisterous-fairy-cede52.netlify.app/`
- ブログ一覧：`https://boisterous-fairy-cede52.netlify.app/blog`

## 確認ポイント

- PC版で横スクロールが出ないか
- Hero見出しが写真に重ならないか
- お悩みカードが途中で切れていないか
- 理由カードと流れカードの幅が崩れていないか
- ヘッダーの「ブログ」から `/blog` へ移動できるか
