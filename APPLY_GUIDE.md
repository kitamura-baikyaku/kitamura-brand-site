# Ver.7.2 反映手順

## 1. ZIPを解凍
`kitamura_brand_v7_2.zip` をダウンロードして解凍します。

## 2. 中身をコピー
解凍したフォルダの中身を、PC内のGitHubフォルダへ上書きコピーします。

例：
`ドキュメント > GitHub > kitamura-brand-site`

## 3. GitHub Desktopで確認
GitHub Desktopを開き、変更ファイルが表示されることを確認します。

## 4. Commit
Summary に以下を入力します。

`Ver7.2 Production Master`

その後、`Commit to main` を押します。

## 5. Push
右上の `Push origin` を押します。

## 6. Netlify確認
数分後に公開URLを開き、PC版・スマホ版で表示を確認します。
